## How to use

Call in function with argument directly

```
.\trs.py buy STM 200 53.6
```

```
python .\trs.py buy STM 200 53.6
```

### function
- buy(symbol, value, price)
- sell(symbol, value, price)
- ui()
- history([symbol])
- hold()
